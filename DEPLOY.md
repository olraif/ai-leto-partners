# Публикация партнерского сайта

Планируемый адрес: `https://partners.ai-leto.ru`.

Порядок запуска после утверждения макета:

1. Создать отдельный GitHub-репозиторий `ai-leto-partners`.
2. Опубликовать содержимое этой папки через GitHub Pages.
3. Указать в GitHub Pages custom domain `partners.ai-leto.ru`.
4. В DNS Timeweb добавить запись для поддомена `partners`.
5. Дождаться выпуска HTTPS-сертификата и проверить все ссылки.

До утверждения макета DNS и публичная версия не создаются.
