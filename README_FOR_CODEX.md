# AI-Лето — партнерский сайт без внутренней информации

Это статический прототип страницы для партнеров. Видимый текст написан так, будто страницу читает сам партнер: лагерь, санаторий, летняя площадка, детский центр, ИП или самозанятый наставник.

## Что внутри

- `index.html` — страница партнеров.
- `assets/css/partners.css` — стили.
- `assets/js/partner-site.js` — список документов, материалов и почтовая заявка.
- `assets/img/` — логотип и постеры.
- `docs/` — оферты, политика ПД, согласие, инструкция, коммерческое предложение.
- `materials/` — постеры, письмо родителям, сообщения и общий архив материалов.

## Что важно для Codex

1. Не добавлять на страницу внутренние формулировки: “MVP”, “demo”, “замените ссылку”, “наша внутрянка”, “настройка backend”, “Prodamus”, “ручная выдача”.
2. Все видимые тексты должны звучать как готовое предложение партнеру.
3. Форму можно подключить к Google Apps Script, Tally, Make, Airtable, собственному backend или другому обработчику.
4. Сайт не собирает данные через форму. Кнопка заявки открывает подготовленное письмо на `info@ai-leto.ru`.
6. Рабочую тетрадь ученика и методичку наставника не размещать на партнерской странице.
7. Материалы для родителей можно размещать: постер, письмо, сообщение в чат.
8. Сохранить цену: 999 ₽ на сайте, 399 ₽ по промокоду партнера, вознаграждение партнера 20% от подтвержденной оплаты по промокоду.

## Рекомендуемый URL

- `partners.ai-leto.ru`
- либо `ai-leto.ru/partners`

## Публикация

Загрузить содержимое папки на поддомен или в папку `/partners`. Для публичного запуска проверить ссылки документов и передать оферты/политику юристу на финальную вычитку.
