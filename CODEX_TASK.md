# Задача для Codex: доделать партнерский сайт AI-Лето без внутренней информации

Нужно доработать статическую партнерскую страницу для AI-Лето. Страница будет доступна партнерам по ссылке, например `partners.ai-leto.ru` или `ai-leto.ru/partners`.

## Главная логика

Партнер читает страницу самостоятельно. Поэтому на странице не должно быть внутренних комментариев, технических пометок и фраз вида:

- “MVP”
- “demo”
- “замените ссылку”
- “не забыть отправить партнеру”
- “наша внутрянка”
- “backend не подключен”
- “ручная выдача”
- “Prodamus”

Все видимые тексты должны быть оформлены как готовое внешнее предложение для партнера.

## Что страница должна объяснять партнеру

1. AI-Лето — готовая 7-дневная AI-активность для детей 10–16 лет.
2. Партнер получает промокод как инструмент привлечения родителей.
3. Обычная цена продукта на сайте — 999 ₽.
4. По промокоду партнера родитель покупает за 399 ₽.
5. Партнер получает 20% от подтвержденной оплаты по промокоду.
6. При цене 399 ₽ вознаграждение партнера — 79,80 ₽ за одну подтвержденную покупку.
7. Возвраты и отмены не учитываются.
8. Рабочая тетрадь и методичка не размещаются в открытом доступе.
9. Для продвижения доступны постеры, письмо родителям и сообщения в чат.

## Блоки страницы

1. Hero:
   - `AI-Лето для партнеров`
   - подзаголовок про промокод, материалы и заявку
   - кнопки: `Подать заявку на промокод`, `Скачать документы`

2. Три выгоды:
   - Родителям — скидка
   - Детям — полезная занятость
   - Партнеру — 20%

3. Как подключиться:
   - ознакомиться с условиями
   - подать заявку
   - получить промокод после проверки
   - использовать материалы для родителей

4. Документы:
   - коммерческое предложение
   - оферта для юрлица / ИП
   - оферта для самозанятого / НПД
   - политика обработки персональных данных
   - согласие на обработку персональных данных
   - инструкция по заявке на промокод

5. Материалы для продвижения:
   - постер для партнера
   - постер для родителей
   - письмо родителям PDF/DOCX
   - сообщение в родительский чат
   - сообщение для руководителей отрядов
   - общий архив материалов

6. Заявка на промокод:
   - заявка отправляется письмом на `info@ai-leto.ru`
   - сайт не собирает и не сохраняет данные
   - в шаблоне письма: организация, город, контактное лицо, аудитория и желаемый промокод
   - телефон и Telegram не запрашиваются

7. FAQ:
   - когда выдается промокод
   - как считается вознаграждение
   - можно ли использовать материалы в родительских чатах
   - можно ли получить полный комплект для массовой рассылки
   - можно ли сделать несколько промокодов

## Технически

- HTML/CSS/JS без тяжелых библиотек.
- Адаптивность для телефона.
- Документы и материалы вынести в конфиг JS, чтобы легко менять ссылки.
- Все ссылки открывать в новой вкладке.
- Кнопка заявки открывает подготовленное письмо на `info@ai-leto.ru`.
- Не добавлять веб-форму и не собирать данные на сайте.
- Не размещать продуктовые PDF: рабочую тетрадь ученика и методичку наставника.

## Готовый результат

Сайт выглядит как готовая страница для партнера, а не как внутренняя инструкция. Партнер может прочитать условия, скачать документы и материалы и отправить заявку по электронной почте.
